מעיין מורדכי — אתר תדמית שמוכן לפריסה ולהמרה לאיקומרס

מה יש כאן?
- index.html — עמוד רספונסיבי מלא (RTL) עם Tailwind מה-CDN.
- assets/ — תקיית תמונות (מקומות־דמה). החליפי לקבצים שלך באותם שמות.

איך מעלים ל‑Vercel:
1) vercel.com → New Project → Manual/Upload → גררי את index.html ואת כל תיקיית assets.
2) שם לפרויקט (maayan-beauty) → Deploy → תקבלי קישור ציבורי.

איך להפוך לאיקומרס בהמשך:
- Shopify Buy Button / Snipcart: מוסיפים סקריפט וקישור “הוספה לסל” בדף.
- או וורדפרס + WooCommerce: מעתיקים את התוכן ומוסיפים חנות.

טלפון/וואטסאפ/מפה כבר בפנים. בהצלחה!
